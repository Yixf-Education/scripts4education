# 天津医科大学（TIJMU）教案模板

## 文件说明
* picins.sty：用于图文混排
* planLL.pdf：理论课教案
* planLL.tex：理论课教案模板
* planSY.pdf：实验课教案
* planSY.tex：实验课教案模板
* TIJMU_plan_LL.cls	理论课教案宏包
* TIJMU_plan_SY.cls	实验课教案宏包

## 文件关系
1. planLL.tex最终输出planLL.pdf，依赖于：
  * picins.sty
  * TIJMU_plan_LL.cls
2. planSY.tex最终输出planSY.pdf，依赖于：
  * picins.sty
  * TIJMU_plan_SY.cls
